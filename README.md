# ExplorADV - Bug Reports

Ce depot permet de signaler des bugs et suggestions pour l'application ExplorADV.

## Comment signaler un probleme

1. Allez dans l'onglet [Issues](../../issues)
2. Cliquez sur "New issue"
3. Selectionnez "Bug Report"
4. Remplissez le formulaire

## Informations utiles a inclure

- Le type de probleme (bug ou suggestion)
- Une description claire du probleme
- Les etapes pour reproduire le bug (si applicable)
- Votre appareil (ex: iPhone 14 Pro)
- Votre version iOS (ex: 17.5)
